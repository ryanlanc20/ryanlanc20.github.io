Personal profile

Hey, there! Let me introduce myself. I am a programmer, pianist, and film lover, with a keen interest in understanding how things work, solving challenges, and building things. My primary interest is in technology, and my passion for it began in early childhood. I started programming in early 2012, initially creating simple desktop applications using VB.net. Shortly thereafter, I started creating web applications using languages such as PHP, and this was when I learned a lot about SQL databases. I continued with this up until 2017, and then it was time for university, where I studied computer science. I learned various programming languages such as C and Java, along with theory in operating systems, distributed systems, and AI. I also worked on lots of complex programming projects (see Education page). After my first degree, I studied a masters in Data Science, with the aim of learning more about AI. Although I am still interested in machine learning, I am primarily focused on becoming a backend developer.
